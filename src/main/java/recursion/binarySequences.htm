Задача  12. «Бинарные последовательности».

Время на тест 1 секунда

 


Условие

Пусть x и y - две бинарных последовательности (элементы последователь-ностей - нули и единицы) размерности n и m соответственно. Сами последовательности x и y можно рассматривать как запись в двоичной форме некоторых двух натуральных чисел. Найти максимальное число z,  двоичную  запись  которого можно получить вычеркиванием цифр как из x,  так и из y. Ответ выдать в виде бинарной последовательности.

 

Входные данные

 

 Входные данные подготовлены в файле input.txt:

1.                 первая строка файла содержит числа   n и m ( 1 ≤ n,m ≤ 1000) ;

2.                 следующие две  строки файла содержат элементы последовательностей x и y соответственно; каждая из   последовательностей задается в виде строки (переменная типа string).

 

Выходные данные

 

Выходные данные  заносятся  в файл output.txt:

  1. первая строка файла содержит число l – максимальное  количество элементов в строке z, задающей двоичную запись  числа;
  2. в следующей строке файла выводится сама бинарная последовательность z  (переменная типа string).

 

 

Пример входных данных

 

11 9

11000100100

001011001

 

Пример выходных данных

 

6

111001
