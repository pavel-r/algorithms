Задача  33. «Цепочка из доминошек».

 


Условие

 

На столе выложены доминошки. Каждая доминошка состоит из двух частей, на каждой из которых изображено некоторое число точек. Доминошки могут выкладываться в ряд одна за другой по следующему правилу: количества точек на примыкающих частях соседних в ряду доминошек должны совпадать. Необходимо определить, можно ли выстроить цепочку, в которой каждая доминошка встречается ровно один раз, причем количества точек на свободных концах доминошек, которые являются крайними в цепочке, должны совпадать.

   

 

Входные данные

  

Входные данные задаются в файле input.txt следующим образом.

·        В первой строке находится количество доминошек N.

·        Следующие N сторок файла содержат данные о доминошках (два числа через пробел): количество точек на верхней и нижней  части доминошки.

 

Выходные данные

 

Выходные данные заносятся в файл output.txt. Если можно составить цепочку из всех доминошек, то первая строка файла содержит сообщение Yes. Если цепочку составить нельзя, то единственная строка выходного файла содержит сообщение No.

 

 

 

Пример входных данных

 

4

3 5

2 3

5 1

1 2

 

Пример выходных данных

 

Yes
