Задача  2.  «Точная сдача». 
(время на тест 3 секунды)

 


Условие

 

У покупателя есть n монет достоинством H1,...,Hn, а у продавца есть m монет достоинством B1,...,Bm. Может ли покупатель приобрести вещь стоимости S  так,  чтобы  у  продавца нашлась точная сдача (если она необходима).

 

Входные данные

 

Входные данные подготовлены в файле input.txt:

1.                 первая строка файла содержит три числа: n m S(1 ≤ n ≤ 100,0 ≤ m ≤ 100, S – натуральное число не превосходящее 2000000000) , разделенные пробелом;

2.                 следующие две строки файла содержат разделенные пробелом монеты покупателя и продавца соответственно( все числа натуральные не превосходящие 100).

 

 

Выходные данные 

 

Выходные данные  заносятся  в файл output.txt, который состоит из единственной строки, Yes  или  No.

 

Пример входных данных

 

3 3 36
12 22 23
12 21 1
 

Пример выходных данных

 

Yes
