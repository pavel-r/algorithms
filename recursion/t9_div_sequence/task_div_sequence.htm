Задача  9. «Выбрать подпоследовательность»  

Время на тест 5 секунда


Условие

 

Из   заданной  последовательности A  целых  чисел длины n необходимо построить подпоследовательность наибольшей длины  такую,  что каждый последующий элемент построенной  подпоследовательности делился бы нацело на предыдущий. Порядок элементов можно менять.

     

Входные данные

 

Входные данные подготовлены в файле input.txt:

1.                 первая строка файла содержит число   n( 0 ≤ n ≤ 11000);

2.                 следующая строка содержит n элементов последовательности A, которые разделены пробелом(все числа целые, по модулю не превосходящие 1000000000).

 

Выходные данные

 

Выходные данные  заносятся  в файл output.txt, который состоит из одной строки:

1.                 l – длина подпоследовательность наибольшей длины  такой,  что каждый последующий элемент построенной  подпоследовательности делится нацело на предыдущий элемент;

 

 

Пример входных данных

 

8

1 2 4 8 16 32 64 100

 

Пример выходных данных

 

7
