Задача  3 . «Проход по матрице».
(время на тест 1 секунда)

 


Условие

 

Задана  матрица  натуральных чисел A из n строк и m столбцов. За каждый проход через клетку с индексами (i,j) взимается штраф A[i,j]. Необходимо с минимальным штрафом пройти  из  какой-либо  клетки  1-ой  строки  в n-ую строку, при этом из текущей клетки  (i,j) можно перейти в любую из 3-х соседних, стоящих в строке с номером на 1 большем: (i+1,j-1) (i+1,j) (i+1,j+1), (если они существуют).

 

 Входные данные

 

   Входные данные подготовлены в файле input.txt:

1.                 первая строка файла содержит размеры матрицы n и m (2 ≤ n ≤ 200, 1 ≤ m ≤ 1000);

2.                 следующие n строк файла содержат элементы матрицы штрафов A по строкам, в строке числа разделяются пробелами (все числа натуральные не превосходящие 1000000).

 

Выходные данные 

 

Выходные данные  заносятся  в файл output.txt, который состоит из одной строки:

1.                 минимальный штраф P.

 

 

 

Пример входных данных

 

2 2

1 1

2 3

 

Пример выходных данных

 

3
